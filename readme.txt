1. install docker
2. install sandbox
	in terminal
		git clone https://github.com/algorand/sandbox.github
3. run testnet
	move to sandbox directory
	in terminal run below for the first time
		./sandbox up testnet
  next time run below
    ./sandbox up
	wait for a while



4. install nodejs v16
	npm install

5. to test backend
	cd src
	node test-backend

6. to test frontend
	npm start
